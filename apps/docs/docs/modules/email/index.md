# Email module

The email module connects backend email service behavior with templates from the
email package.

## Package map

| Package | What it owns |
| --- | --- |
| `@m5kdev/backend` | Email module registration and email service behavior. |
| `@m5kdev/email` | React Email templates and template build tooling. |
