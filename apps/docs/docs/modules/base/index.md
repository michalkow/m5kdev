# Base module

The base module contains shared backend abstractions, service procedure helpers,
permission grants, actors, DTOs, and shared schema primitives.

## Package map

| Package | What it owns |
| --- | --- |
| `@m5kdev/commons` | Base constants and schemas. |
| `@m5kdev/backend` | Base repositories, services, permission services, actors, grants, procedure helpers, and module contracts. |
