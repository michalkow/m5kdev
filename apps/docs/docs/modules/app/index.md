# App shell module

The app shell module contains shared app configuration, tRPC query providers, and
the reusable application shell UI.

## Package map

| Package | What it owns |
| --- | --- |
| `@m5kdev/frontend` | App config provider, app config hook, and app tRPC query provider. |
| `@m5kdev/web-ui` | App loader, shell, sidebar, sidebar content, and sidebar user components. |
