# Commons package

`@m5kdev/commons` contains shared constants, schemas, types, and utilities used by
backend and frontend packages.

## Use it for

- Cross-package Zod schemas.
- Shared constants such as file type definitions.
- Type contracts that must stay identical across backend and frontend code.

## Module docs

Start with these module pages:

- [File](/modules/file)
- [Auth](/modules/auth)
- [Billing](/modules/billing)
- [Table](/modules/table)
- [Workflow](/modules/workflow)

## Package rule

Put shared contracts here before wiring backend validation or frontend UX around
them.
